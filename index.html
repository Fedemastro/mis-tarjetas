Ingresá tu API Key de Google Sheets para sincronizar datos entre dispositivos.
1

Andá a console.cloud.google.com, creá un proyecto nuevo y habilitá la Google Sheets API.

2

En APIs & Services → Credentials, creá una API Key y una OAuth 2.0 Client ID (tipo Web App). Agregá tu URL de GitHub Pages como origen autorizado.

3

Creá una Google Spreadsheet nueva y copiá el ID de la URL (el string largo entre /d/ y /edit).
